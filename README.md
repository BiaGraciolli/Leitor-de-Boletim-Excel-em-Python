# Projeto do 3° Bimestre: Sistema de Geração de Páginas Web e PDFs de Notas por Leitura de Excel

# Descrição do projeto: 
Este projeto foi desenvolvido em **Python**. 
Permite a leitura de notas de planilhas .xlsx e cria páginas HTML individuais para cada aluno, arquivo PDF com notas e média, exibição dos dados de cada aluno por meio da biblioteca PrettyTable.
O programa identifica automaticamente todas as disciplinas e destaca notas abaixo de 6,0 em vermelho e notas a partir de 6,0 em azul. Notas não exibidas são declaradas como "NÃO INFORMADA".
O usuário deve informar o caminho completo do arquivo e o nome da planilha. A planilha deverá possuir o código, aluno, turma, todas as matérias e a média como colunas.

-----------------------------------------------------
Tecnologias:
- **Python 3.13**
- **HTML e CSS**
- Bibliotecas:
  - `pandas` (para manipulação da planilha excel)
  - `prettytable` (para exibir a tabela no console)
  - `ReportLab` (para gerar PDFs)
